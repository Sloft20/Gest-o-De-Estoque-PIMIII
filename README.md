# Gerenciador de Estoque - PIM III UNIP

Sistema de gerenciamento para pequenos mercados desenvolvido como parte do Projeto Integrado Multidisciplinar (PIM III) do curso de ADS.

## 🚀 Tecnologias Utilizadas

### Back-end
- **Linguagem:** C#
- **Framework:** ASP.NET Core Web API 8.0
- **Conceitos de POO:** Encapsulamento, Herança e Polimorfismo.
- **Armazenamento:** Banco de dados em memória (estático).

### Front-end
- **Tecnologias:** HTML5, CSS3, JavaScript Vanilla.
- **Responsividade:** Mobile First (Flexbox & Grid).
- **Acessibilidade:** Integração com VLibras.
- **Ícones:** FontAwesome.

## 🛠️ Como rodar o projeto

### 1. Pré-requisitos
- Ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado.
- Um navegador moderno (Chrome, Edge, Firefox).

### 2. Rodando o Back-end (API)
1. Abra o terminal na pasta `Back-End-CSharp/GerenciadorEstoqueAPI`.
2. Execute o comando:
   ```bash
   dotnet run

O terminal informará a URL (ex: http://localhost:5164). Certifique-se de que esta URL é a mesma configurada no arquivo app.js do front-end.

## 3. Rodando o Front-end
Navegue até a pasta Front-End-Web.

Abra o arquivo index.html em qualquer navegador.
